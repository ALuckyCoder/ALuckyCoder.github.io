centos7中安装ntfs-3g

  1. 加源
    wget -O /etc/yum.repos.d/epel.repo http://mirrors.aliyun.com/repo/epel-7.repo
  2. 安装
    yum install ntfs-3g

centos7中yum升级被PackageKit锁定

vim /etc/yum/pluginconf.d/langpacks.conf

将第一行：enable=1改为enable=0
然后重新启动系统

centos7下自动开机挂载

vim /etc/fstab
在最后加上配置信系如下
/dev/sda3 /mnt/data ntfs default 0 1

第一列是第一列是分区名，第二列是需要挂载的路径,第三列是该分区的文件格式,然后注意挂载目录的权限设置即可

小知识点记录

linux下的垃圾回收站是/dev/null

关于RANDOM：RANDOM变量内容介于0~32767之间执行 echo $RANDOM 时，系统就会主动的随机取出一个介于0~32767的数值。可以用 declare -i number=$RANDOM*10/32768 设置number取0~9之间的数值
用set可以观察所有变量（包括环境变量和自定义变量）

bash默认的组合键如下

组合按键 执行结果
Ctrl + C 终止目前的命令
Ctrl + D 输入结束（EOF）
Ctrl + M 就是Enter
Ctrl + S 暂停屏幕的输出
Ctrl + Q 恢复屏幕的输出
Ctrl + U 在提示符下，将整行命令删除
Ctrl + Z 暂停目前的命令

bash下常用的通配符

符号 意义
* 代表0个到无穷多个任意字符
? 代表一定有一个任意字符
[] 代表中括号内的任意一个字符，如[abcd]表示的是一定有一个字符，可能是a,b,c,d中的任何一个
[-] 减号在中括号内，代表在编码内的所有字符，如：[0-9]代表0到9之间的所有数字, 因为数字的语系编码是连续的
[^] 表示反向选择，例如[^abc]代表一定有一个字符，这个字符不是a,b,c中的任何一个

centos7中一些文件的含义

/etc/shells：提供可用shell的相关信息
/lib/modules/${uname -r}/kernel：核心的板块目录
/dev/random：随机数生成器文件
~/.bash_history：记录历史命令
/etc/issue：记录存储进系统画面显示的相关信息
/etc/motd：记录进系统后提示给用户的相应信息
